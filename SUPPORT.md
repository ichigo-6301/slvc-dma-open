# Support Boundary

This repository supports public-source questions that include a reproducible
command and sanitized evidence. It does not provide an SLA, commercial-tool
license support, PDK support, board bring-up support, generated-IP support, or
private integration debugging.

Use a reproducible issue for RTL, documentation, or public-flow problems. Use
the security channel for vulnerabilities. The documented limitations and
nonclaims are part of the supported public boundary.
